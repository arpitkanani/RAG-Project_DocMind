Machine learning is a subset of artificial intelligence.
It enables computers to learn from data automatically.
ML algorithms improve their performance over time.
Common ML types include supervised and unsupervised learning.

Neural networks are inspired by the human brain structure.
They consist of layers of interconnected nodes called neurons.
Deep learning uses multiple layers to extract complex features.
CNNs are used for image recognition tasks specifically.

Python is the most popular language for data science.
Libraries like NumPy and Pandas help with data manipulation.
Scikit-learn provides simple machine learning implementations.
TensorFlow and PyTorch are used for deep learning projects.

Football is a popular sport played worldwide by millions.
The FIFA World Cup is held every four years internationally.
Players need stamina, skill and teamwork to win matches.
Lionel Messi is considered one of the greatest footballers.
